# Configuração do Frontend

O frontend utiliza **React 18** com **TypeScript**.

## Passos

1. Instale as dependências:  
   `npm install`
2. Configure variáveis de ambiente (`.env`)
3. Rode o projeto:  
   `npm start`

## Principais tecnologias

- React 18
- Redux Toolkit
- Material-UI v5
- Recharts (gráficos)
- React Hook Form (formulários)
