# Autenticação e Usuários

## Recursos

- Cadastro com email e senha (validação forte)
- Login seguro (email/senha, Google OAuth, Apple ID)
- 2FA opcional via SMS/app
- Recuperação de senha com link temporário
- Perfil do usuário: nome, foto, telefone, preferências
- Configurações de privacidade

## Segurança

- JWT com refresh token
- Limite de tentativas de login
- Criptografia TLS 1.3, AES-256
